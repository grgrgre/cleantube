# 🛡️ CleanTube

A privacy-first, context-aware content filter for YouTube that eradicates clickbait, shock content, and algorithmic noise, leaving only the topics you actually want to focus on.

---

### 🚀 How It Works
The extension monitors your YouTube feed dynamically. Instead of relying on rigid, outdated keyword matching, it aggregates video titles into **batches** and runs them through a context-aware smart engine to evaluate relevance.

> **Context Example:** If your focus topic is **"Rust"**, a video titled *"Writing a high-performance compiler"* stays. A video titled *"OMG I ALMOST CRASHED THE PRODUCTION SERVER! 😱"* gets wiped from your screen instantly.

---

### ✨ Features
* **Smart Batching:** Evaluates 10–20 videos in a single engine call, bypassing rate limits and ensuring smooth scrolling.
* **100% Free & Private:** Runs entirely client-side using your own free engine key. No third-party servers, data tracking, or hidden subscriptions.
* **Manifest V3 Compliant:** Built using Google's latest secure extension standards.
